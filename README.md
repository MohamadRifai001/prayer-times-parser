# Prayer Times Parser

This function extracts prayer times from a semi-structured HTML source that does not provide a formal API.

## Highlights
- Parses inconsistent HTML data using Cheerio
- Handles Hijri and Gregorian date extraction
- Extracts multiple prayer times and formats them
- Includes timeout handling and error recovery
- Supports fetching next month's data

This was used in a production application to automate monthly prayer time updates.
